# Delivery Service Web App

Тестовое задание на позицию Разработчик C# .Net. 
Приложение для управления заказами на доставку.

## Стек технологий

*   **Backend:** ASP.NET 9 (Web API), Entity Framework Core.
*   **Frontend:** React 18 (Vite), React Router v6, Axios.
*   **База данных:** SQLite.
*   **Архитектура:** 
    *   Разделение ответственности: Controllers, Services, Repositories (EF Context).
    *   Использование DTO для изоляции моделей БД от контрактов API.
    *   Валидация данных на уровне моделей.

## Функциональность

1.  **Создание заказа:** Форма с валидацией всех полей. Номер заказа генерируется автоматически на бэкенде.
2.  **Список заказов:** Отображение всех созданных заказов в виде сетки.
3.  **Детальный просмотр:** Отдельная страница для просмотра полной информации о заказе в режиме чтения.
4.  **Дополнительно:** 
    *   Реализовано полное редактирование заказа с предзаполнением формы.
    *   Реализовано удаление заказов.
5.  **UI/UX:** Интерфейс выполнен в фиолетовой цветовой гамме Visual Studio Style.

## Подготовка окружения перед запуском

Для успешного запуска проекта "одной кнопкой" (F5) убедитесь, что выполнены следующие шаги:

1. **Установка ПО:**
   - Установите [.NET 9.0 SDK](https://dotnet.microsoft.com/en-us/download/dotnet/9.0)
   - Установите [Node.js (LTS версия)](https://nodejs.org/)

2. **Настройка PowerShell (только для Windows в случае отключенного выполнения сценариев):**
   Если при запуске проекта появляется ошибка "выполнение сценариев отключено", разрешите выполнение локальных сценариев (нужно для работы Vite). Откройте PowerShell и выполните:
   ```powershell
   Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
   ```

## Настройка

Проект настроен на одновременный запуск фронтенда и бэкенда из Visual Studio.

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/AntonMikityuk/DeliveryService.git
    ```
2.  **Подготовка фронтенда:**
    Перейдите в папку `frontend` и установите зависимости:
    ```bash
    cd frontend
    npm install
    ```
## Запуск
Проект настроен на одновременный запуск Frontend и Backend. Выполните следующие действия:

1. Откройте файл решения `DeliveryService.slnx` в **Visual Studio 2022**.
2. Нажмите правой кнопкой мыши на **Решение "DeliveryService"** в Обозревателе решений и выберите **Свойства**.
3. Перейдите в раздел **Общие свойства** -> **Настроить запускаемые проекты**.
4. Выберите вариант **"Несколько запускаемых проектов"**.
5. В списке проектов установите следующие параметры:
   - **DeliveryService**: действие — **Запустить**.
   - **frontend_vs**: действие — **Запустить**.
   - Целевой объект оставьте пустым.
6. Нажмите "ОК" и запустите проект клавишей **F5**.

Бэкенд запустится на порту `7068` (HTTPS), фронтенд — на порту `5173`. 
Файл базы данных `delivery.db` уже содержит миграции и готов к работе.
